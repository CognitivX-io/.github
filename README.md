# .github

Organization-wide defaults for the [CognitiveX](https://github.com/CognitivX-io)
organization.

GitHub serves the files in this repository as fallbacks for every repo in the
org that doesn't define its own. Edit them here once and they apply everywhere.

| File | Purpose |
|------|---------|
| [`profile/README.md`](profile/README.md) | The org profile shown at github.com/CognitivX-io |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Community standards (Contributor Covenant) |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | How to contribute |
| [`SECURITY.md`](SECURITY.md) | How to report a vulnerability |
| [`SUPPORT.md`](SUPPORT.md) | Where to get help |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) | Bug report & feature request forms |
| [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) | Default PR template |

A repository's own file always wins over the default defined here.
