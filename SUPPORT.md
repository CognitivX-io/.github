# Support

Need help with CognitiveX, iCog, cogix, or cogx? Here's where to go.

## Questions and how-to

- **Docs** — [cognitivx.io/docs](https://cognitivx.io/docs)
- **Discussions** — open a thread in the relevant repository's Discussions tab.

## Bugs

Open an issue in the repository where the bug lives, using the **Bug report**
template. Include your version, environment, and steps to reproduce.

## Feature requests

Open an issue using the **Feature request** template, or start a Discussion if
you want to talk through the idea first.

## Account, billing, and product help

Email **contact@cognitivx.io** for anything account- or billing-related.

## Security

For vulnerabilities, **do not** open a public issue — follow
[SECURITY.md](SECURITY.md).
